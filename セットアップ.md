
## フロントエンド

```bash
winget install -e --id OpenJS.NodeJS.LTS

cd ./frontend/
npm install
npm run build
```

スタート

```bash
cd ./frontend/
npm run preview
```

## バックエンド

```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

cd ./backend/
uv sync
```

スタート

```bash
cd ./backend/
uv run uvicorn app.main:app
```
